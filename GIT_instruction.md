## Списки

Чтобы добваить ненумерованные списки, необходимо пункты выделить звёздочкой (*).

Например, вот так:

* Элемент 1
* Элемент 2 
* Элемент 3

Чтобы добавить нумерованные списки, необхоодимо пункты просто пронумеровать.

Например, вот так :
1. Первый пункт
2. Второй пункт
